# Deskemoji v0.2 设计文档

## 概述
Deskemoji 是一个 Windows 桌面小程序，显示动态 emoji 表情。v0.2 增加完整的交互功能和设置系统。

## 新增功能

### 1. 右键菜单系统

**菜单结构：**
```
手动选择 >
  🙂 开心
  😢 难过
  😡 生气
  😴 困倦
  🤔 思考
  🥵 热
  💀 崩溃
  🌙 晚安
─────────
自动模式 ✓
─────────
设置
开机启动
─────────
退出
```

**交互逻辑：**
- 左键点击 emoji → 触发弹跳动画
- 右键点击 → 显示菜单
- 选择手动表情 → 禁用自动模式，固定显示该表情
- 点击"自动模式" → 切换自动/手动

### 2. 设置窗口

弹出式设置窗口，包含：

| 设置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| 自动模式 | 开关 | 开 | 是否根据时间/系统状态自动变化 |
| 开机启动 | 开关 | 关 | 开机自动启动 |
| 更新频率 | 下拉 | 2秒 | 系统状态检测频率 |
| CPU阈值 | 滑块 | 80% | 触发🥵的CPU使用率 |
| 内存阈值 | 滑块 | 90% | 触发💀的内存使用率 |
| 空闲时间 | 输入 | 5分钟 | 触发😴的空闲时间 |
| 窗口透明度 | 滑块 | 100% | 背景透明度 |
| 窗口大小 | 输入 | 120px | emoji窗口大小 |

### 3. 配置文件系统

**配置文件路径：** exe 同目录下的 `config.json`

**首次运行：** 自动生成默认配置

**配置格式：**
```json
{
  "auto_mode": true,
  "startup": false,
  "update_interval_secs": 2,
  "cpu_threshold": 80.0,
  "memory_threshold": 90.0,
  "idle_threshold_secs": 300,
  "opacity": 1.0,
  "window_size": 120
}
```

## 技术实现

### 新增依赖
```toml
serde = { version = "1.0", features = ["derive"] }
serde_json = "1.0"
```

### 新增模块
```
src/
├── main.rs
├── monitor.rs
├── emoji.rs
├── renderer.rs
├── config.rs      # 新增：配置管理
├── menu.rs        # 新增：右键菜单
└── settings.rs    # 新增：设置窗口
```

### 模块职责

**config.rs:**
- Config 结构体定义
- 加载/保存配置
- 首次运行创建默认配置

**menu.rs:**
- 创建右键菜单
- 处理菜单事件
- 管理手动/自动模式切换

**settings.rs:**
- 创建设置窗口
- 绑定设置控件
- 实时保存设置

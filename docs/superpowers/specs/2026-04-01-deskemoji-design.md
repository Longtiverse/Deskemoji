# Deskemoji 设计文档

## 概述
Deskemoji 是一个 Windows 桌面小程序，在桌面上显示一个动态的小黄脸表情，根据不同情景与用户互动。

## 技术选型
- **框架**：Tauri（Rust + Web技术）
- **优势**：轻量（<10MB）、性能好、资源占用低

## 界面设计

### 视觉风格
- **极简风格**：透明背景，无边框
- emoji 像贴纸一样漂浮在桌面
- 支持拖拽移动位置

### 动态效果
- **微动效果**：轻微浮动动画（上下浮动）
- 表情变化时有平滑过渡
- 不干扰用户正常工作

## 情景系统

| 情景 | 表情 | 触发条件 |
|------|------|----------|
| 早晨 | 🙂 | 6:00-10:00 |
| 上午 | 😊 | 10:00-12:00 |
| 午后 | 🤗 | 12:00-14:00 |
| 下午 | 😌 | 14:00-18:00 |
| 晚上 | 🌙 | 18:00-23:00 |
| 深夜 | 😪 | 23:00-6:00 |
| 工作中 | 🤔 | 检测到键盘/鼠标活动 |
| 空闲 | 😴 | 5分钟无操作 |
| CPU高 | 🥵 | CPU > 80% |
| 内存高 | 💀 | 内存 > 90% |
| 通知 | 😲 | 收到系统通知 |
| 充电中 | 😎 | 连接电源 |
| 电量低 | 😰 | 电量 < 20% |

## 交互方式

### 鼠标操作
- **左键点击**：手动切换表情/显示当前状态
- **左键拖拽**：移动位置
- **右键点击**：弹出菜单

### 右键菜单
- 设置
- 开机启动（开关）
- 退出

## 优先级
- 第一版：基础显示 + 时间情景 + 系统状态
- 后续：通知提醒 + 更多动画效果 + 自定义表情

## 验证标准
- [ ] 程序启动后显示emoji
- [ ] 表情随时间变化
- [ ] CPU/内存高时显示对应表情
- [ ] 可拖拽移动位置
- [ ] 右键菜单可退出
- [ ] 打包体积 < 15MB
